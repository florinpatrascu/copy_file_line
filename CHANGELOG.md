# Changelog

All notable changes to the CopyFileLine addon will be documented in this file.

## [1.1.1] - 2026

### Added

- Copy script path and line number with customizable keyboard shortcuts
- Two copy modes: resource path (res://) and filesystem path
- Configurable shortcuts in Editor Settings
- Default shortcuts:
  - **Ctrl+Shift+C**: Copy as resource path (res://)
  - **Ctrl+Shift+Alt+C**: Copy as filesystem path

### Features

- Adds menu items to Tools menu for manual copying
- Automatically copies current cursor line number
- Supports both Godot resource paths and absolute filesystem paths
- Shortcuts persist in Editor Settings for user customization
